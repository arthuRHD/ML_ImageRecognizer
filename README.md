# ATA Project

A Django application that classify sended pictures with TensorFlow and store them with MangoDB

## Install

```sh
cd projet-ata
pip install -r requirements.txt
```

## Usage

Launch the django app

```sh
cd projet-ata
python3 manage.py runserver 0.0.0.0:8000
```
